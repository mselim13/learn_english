## 🇬🇧 English

### About

**Learn English** is a Flutter mobile app built to help Turkish speakers learn English step by step. It offers various exercises to improve vocabulary, listening, writing, and speaking skills.

### Features

| Feature | Description |
|--------|-------------|
| 📚 **Lessons** | Level-appropriate lesson content |
| 📖 **Vocabulary Book** | Save words, study with flashcards |
| 🎯 **Word Quiz** | Test your vocabulary |
| 🎧 **Listening Exercises** | Improve listening comprehension |
| 🗣️ **Speaking Practice** | Pronunciation and speaking drills |
| ✍️ **Writing Practice** | Strengthen your writing skills |
| 🔤 **Flashcard** | Quick vocabulary review |
| 🌐 **Translation** | Word and sentence translation |
| 📊 **Statistics** | Track your progress |
| 🏆 **Badges** | Celebrate your achievements |
| 🗺️ **Level Roadmap** | Visualize your learning journey |
| ⚙️ **Settings** | Notifications, daily goals, and more |

### Installation

**Requirements:**
- Flutter SDK (^3.10.4)
- Dart 3.10+

```bash
# Clone the repository
git clone https://github.com/mselim13/learn_english.git
cd learn_english

# Install dependencies
flutter pub get

# Run the app
flutter run
```
